# Project Brief: Cuboid

**Protocol**: Alpha Prime
**Objective**: Build a local-first, professional code editor with Prism-level interaction quality, hard-square industrial aesthetics, and a provider-agnostic AI layer.

## Core Technologies
- **Frontend**: React, Vite, TypeScript
- **Styling**: Tailwind CSS (Hard-square industrial minimal system)
- **Editor**: Monaco Editor (Primary target per PLAN.md)
- **Data**: Local-first storage through adapter boundaries (`DataStore`, `AuthService`, `CryptoVault`, `CompileService`)
- **AI**: Abstracted Multi-Provider SDK

## Architecture Principles
1.  **Local-First Verification**: Phase 1 must run entirely without a backend.
2.  **Aesthetic Excellence**: Strict square-edge system, restrained contrast, and coherent interaction states.
3.  **Sovereignty**: Future-proofed for Zero-Knowledge (Client-side encryption).
4.  **Controlled Agentic Behavior**: Assistant/agent autonomy is user-scoped and explicit, not implicit.
