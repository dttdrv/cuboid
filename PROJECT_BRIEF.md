# Project Brief: Cuboid

**Protocol**: Alpha Prime
**Objective**: Build a local-first, premium code editor with "Prism"-level aesthetics and a provider-agnostic AI layer.

## Core Technologies
- **Frontend**: React, Vite, TypeScript
- **Styling**: Tailwind CSS (Premium/Glassmorphism)
- **Editor**: Monaco Editor (Primary target per PLAN.md)
- **Data**: LocalStorage (Phase 1) -> Convex (Phase 2)
- **AI**: Abstracted Multi-Provider SDK

## Architecture Principles
1.  **Local-First Verification**: Phase 1 must run entirely without a backend.
2.  **Aesthetic Excellence**: "Premium" feel, glassmorphism, micro-animations.
3.  **Sovereignty**: Future-proofed for Zero-Knowledge (Client-side encryption).
